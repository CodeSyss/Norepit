Proyecto Destacado

Sistema de Recomendación de Cursos

Desarrollé un sistema de recomendación de cursos utilizando Python y Machine Learning para una plataforma educativa. El sistema analiza las preferencias de los usuarios y el historial de cursos para ofrecer recomendaciones personalizadas.

  • Tecnologías: Python, scikit-learn, Flask, PostgreSQL
  • Funcionalidades: Filtrado colaborativo, filtrado basado en contenido, evaluación de recomendaciones
  • Resultados: Aumento del 20% en la tasa de inscripción de cursos
